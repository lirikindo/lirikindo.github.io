4. YOU HAVE BEEN WAITING HERE FOR TWO HOURS. Make the negative sentence​


kalimat di atas merupakan present perfect continuous tense

rumusnya

(+) S + has/have + been + Ving + O

(-) S + has/have + not + been + Ving + O

(-) has/have + S + been + Ving + O

Penjelasan:

(+) YOU HAVE BEEN WAITING HERE FOR TWO HOURS

(-) YOU HAVE not BEEN WAITING HERE FOR TWO HOURS

(?) have you BEEN WAITING HERE FOR TWO HOURS?

Penjelasan:

present tense

(+) S + V1

(-) S + do/does + not + V1

(?) Do/does + S + V1 + ?

I, he, she, it => does

we, you, they => do

present perfect tense

(+) S + has/have + V3

(-) S + has/have + not + V3

(?) has/have + S + V3 + ?

he, she, it => has

I, you, we, they => have

future tense

(+) S + will + V1

(-) S + will + not + V1

(?) Will + S + V1 + ?
