27. #Soal Materi Sistem Persamaan Linear Dua Variabel Kelas 9 smt 2# Himpunan Penyelesaian dari X+Y=7 dan X-Y= -5 adalah...???


Eliminasi,

x + y = 7
x - y = -5
________ -
2y = 12
y = 12/2
y = 6

x + y = 7
x + 6 = 7
x = 7 - 6
x = 1

HP = {x, y}
HP = {1, 6}

28. tolong ajarin no 7...kelas X smt gasalmakasih


|4x - 3| = |x - 7|
|4x - 3| - |x - 7| = 0
(4x - 3)² - (x - 7)² = 0
(4x - 3 + (x - 7))(4x - 3 - (x - 7)) = 0
(4x - 3 + x - 7)(4x - 3 - x + 7) = 0
(5x - 10)(3x + 4) = 0

5x - 10 = 0
5x = 10
x = 2

dan

3x + 4 = 0
3x = -4
x = -4/3

HP = {-4/3, 2}
