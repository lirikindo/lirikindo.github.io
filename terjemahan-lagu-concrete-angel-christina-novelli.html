14. fungsi novell netware


Novel netware adalah sebuah network operating system (NOS)  yang umum digunakan dalam komputer IBM PC atau kompatiblenya. 
jadi secara garis besar novel netware berfungsi sebagai operasi sistem pada sebuah jaringan khusus komputer IBMNovell Netware merupakan sebuah system jaringan yang didirikan oleh Novell corporation, yang mana jaringan NOVELL ini lebih baik ketika dipakai oleh komputer yang sifatnya untuk pribadi (komputer PC). Dan Novell ini juga menyediakan beberapa layanan untuk saling berbagi data file, mesin printer atau jenis layanan lainnya. Novell Netware, sekitar pada tahun 1990 an pertengahan mengalami suatu persaingan antar perancang teknologi dari Microsoft dengan melakukan sebuah pengenalan terhadap teknologi yang telah mereka buat seperti windows NT server yaitu merupaka sebuah produk milik Microsoft. Memang system jaringan Novell Netware sedikit yang mengenalnya, bahkan mendengarnya saja sudah terdengar asing bagi telinga. Masyarakat umum terutama masyarakat yang masih awam sebagian besar hanya mengenal beberapa saja jenis operasi sistem seperti halnya windows atau pun seperti linux yang dipakai sebagai media system operasi mereka. Dalam suatu jaringan system komputer, pastilah memiliki Kekurang / kelemahan atau keunggulan yang dimilikinya, begitu juga dengan jaringan system Novell Netware ini. Nah disini akan saya ulas beberapa kelebihan serta kekurangan yang dimiliki oleh Novell Netware. Berikut ulasannya. Kekurangan / kelemahan yang dimiliki oleh Novell Netware, diantaranya: Masih kalah unggul dengan perusahaan Microsoft yang mana mereka telah membuat rencana yang lebih canggih dan lebih baik guna untuk mempermudah kebutuhan para user terutama bagi user yang bekerja didepan komputer, lain halnya dengan Novell, meskipun NOVELL pernah booming pada awal tahun pembuatannya, akan tetapi pada tahun yang semakin lama semakin canggih ternyata NOVELL masih kalah unggul dengan perusahaan Microsoft. NOVELL server pada saat itu adalah server yang dipakai sebagai tempat yang menyediakan database yang diperlukan, akan tetapi server tidak menjalankan proses komputasi melainkan yang menjalankannya adalah komputer client. Nah hal tersebut sudah tidak lagi digunakan karena tidak diperlukan juga tidak layak lagi sebagai tempat untuk memenuhi kebutuhan. Berbeda halnya dengan teknologi yang dikembangkan microsoft yang sangat praktis dan mudah dilakukan untuk pemenuhan kebutuhan. Kelebihan yang dimiliki oleh Novell Netware, diantaranya: Novell pada tahun pembuatan pertengahannya telah merancang NOVELL 5 yang mana NOVELL server tersebut meningkatkan kinerja pada tingkat skala yang lebih baik, artinya NOVELL telah membuat sebuah fitur yang tidak dimiliki oleh Windows NT yaitu fitur storage servis dimana sebuah file yang ditampung didalamnya dengan berbagai ukuran file yang hampir tidak ada batasannya. Dalam hal komunikasi, pemakaian NOVELL server ini mempermudah proses komunikasi antara komputer PC sehingga ketika sedang terjadi komunikasi melalui NOVELL server ini akan lebih cepat. Sebuah Informasi Defini dari Novell Operating Sistem (Novell Netware) telah saya jabarkan diatas, dan disini saya harap dapat dipelajari dan dipahami akan ilmu komputer itu sangatlah luas, tidak hanya perangkat komputer dan isi komputer saja yang dipelajari, akan tetapi software yang begitu tak terbatas pun juga harus diketahui, agar ilmu kita terutama pada user yang menjalankan, akan lebih baik lagi apabila ilmu tentang komputernya dalam luas.

PERSIAPAN  &  INSTALLATION  NOVELL NETWARE  5.X
Persiapan Installation Novell Netware  5.x :
1.        Menyiapkan Software dan System yang dibutuhkan.
2.       Menyiapkan Network untuk Server Netware 5.0
3.       Menyiapkan komputer untuk server installasi
Installation  Novell  Netware  5.x :
1.        Memilih type dari installation
2.       Memilih  setting regional.
3.       Memilih module yang support untuk platform dan adaptor storage.
4.      Memilih device storage dan board network
5.       Membuat partisi netware dan volume sys.
6.      Membuat penamaan server Netware 5
7.       Install server Netwate 5 file system
8.      Install protokol jaringan
9.      Set  Server  zona waktu
10.    Set  NDS
11.     Lisence
12.    Memilih product lain yang akan diinstall
INSTALLASI  SERVER NETWARE 5.0
I.        Perangkat yang diperlukan untuk membangun Server Novell Netware 5.0  :
a.       CPU setara pentium atau diatasnya
b.      VGA  dengan resolusi tinggi.
c.       Hard Disk kapasitas 550 MB atau lebih (50 MB untuk partisi DOS, 500 MB untuk netware partisi).
d.      Mouse PS/2
e.      CD installer Netware 5.0
f.        Disket License Netware 5.0
g.      Disket Dos 6.2 atau sebelumnya (DOS 7 sudah diincludekan kedalam disket license,  jangan menggunakan DOS yang bawaan dari windows 95, windows 98, windows NT)
h.      Novel Netware  Client untuk DOS dan Windows 3.1x
i.         Konfigurasi NIC  (seperti interupt dan alamat port)
j.         Untuk Netware 4.11  anda perlu mengupgrade  network dengan cara : Update NDS ke NDS 5.99 atau yang lebi tinggi. NDS versi 5.99 sudah tersedia didalam cd netware 5.
II.     Install Novell Server Novell Netware 5.0
1.        Pasanglah NIC, HardDisk, dan CD ROOM pada CPU server
2.       Lakukan booting dengan menggunakan DOS 6.22
3.       Untuk menginstall server melalui CD ROOM terlebih dahulu harus membuat konfigurasi system yang melakukan load terhadap CD ROM, seperti contoh berikut ini :
A:\COPY CON CONFIG.SYS
FILE=40
BUFFERS=30
DEVICE=HIMEM.SYS
DEVICE=HIMEM.SYS
Kemudian simpan file config.sys
Setelah file CONFIG.SYS selesai dibuat, buatlah file AUTOEXEC.BAT yang digunakan untuk menjalankan program memanggil CD ROOM yang sudah definisikan pada CONFIG.SYS sebagai berikut:
A:\copy con autoexec.bat
MSCDEX/D:MTMCD001
Kemudian simpan file autoexec.bat
Catatan : gunakan driver CDROM yang sesuai CDROM anda.
4.      Sebelum memulai melakukan installasi server terlebih dahulu buatlah partisi harddisk dengan pembagian sbb :
Buat  partisi DOS (DOS pentium) minimum 50 MB. Caranya :
A:\FDISK
Menampilkan sbb:
Choose one of following :
1. Create DOS partition or Logical DOS Drive.
2. Set  active partition
3. Delete partition or Logical DOS Drive
4. Display partition Information.
Pilih pilihan nomor 1, tekan enter yang akan menampilkan pilihan sbb:
Choose one of following :
1. Create primaty DOS partition.
2. Create Extended DOS partition.
3. Create Logical DOS Drive (s) in the extended DOS partition.
Pilih pilihan nomor 1 dan selanjutnya tampil  :
Create primary DOS partition.
Current fixed disk drive 1.
Do you want the maximum available size for a primary DOS Partition and make the partition active (Y/N)?
Pilih N dan tentukan besarnya, besar partisi cukup 50MB dan sisanya untuk partisi Novell, yang akan dibuat pada proses instalasi.
Tekan ESC untuk kembali ke menu utama FDISK
Tandai partisi yang anda buat dengan menentukan sel aktive partisi yaitu pada pilihan 2
Setelah pembuatan partisi tersebut komputer meminta untuk melakukan booting ulang. Pada proses booting ini tetap gunakan Disk Boot DOS 6.22 yang sudah anda buat diatas tadi.
Formatlah 50 MB partisi DOS diatas dengan menambahkan system A:\FORMAT c:/s
Boot ulang dan tetap menggunakan disket DOS 6.22.
Lakukan proses installation server dari CD_ROM.
