33. perbandingan uang via terhadapa uang valen adalah 3 : 4. jika selisih uang mereka Rp 2.500.000, maka jumlah uang via dan uang valen adalah jawab cepet cukk!


jumlah uang mereka
=(3+4)/(4-3)x2.500.000,00
=7/1x2.500.000,00
=17.500.000,00
maaf kalau salah Selisih perbandingan :
4-3=1

uang Valen :
[tex] = \frac{4}{1} \times 2.500.000 \\ = \frac{10.000.000}{1} = 10.000.000[/tex]

uang Via :
[tex] = \frac{3}{1} \times 2.500.000 \\ = \frac{7.500.000}{1} = 7.500.000[/tex]

Jumlah uang Via dan Valen :
[tex] = 10.000.000 + 7.500.000 \\ = 17.500.000[/tex]
Jadi,jumlah uang Via dan Valen adalah Rp17.500.000.
